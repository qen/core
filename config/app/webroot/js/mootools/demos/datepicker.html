Variable navigation options

Calendar includes 3 navigation options: traditional navigation where left and right arrows allow the user to navigate by month (default); improved navigation where the user can navigate by month or year; and static navigation, which is actually no navigation at all. I wanted navigation to be highly usable without complicating the interface with popup selects, text inputs or buttons. myCal1 = new Calendar({ date1: 'd/m/Y' });
myCal2 = new Calendar({ date2: 'd/m/Y' }, { classes: ['alternate', ...], navigation: 2 });
myCal3 = new Calendar({ date3: 'd/m/Y' }, { navigation: 0 });
