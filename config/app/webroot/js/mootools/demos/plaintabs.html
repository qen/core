PlainTabs Sample Implementation

PlainTabs is another tabbed content extension for MooTools. The goal was to produce an extension that did not rely on CSS classes to tie together functionality. You should be able to produce tabbed content with cleaner markup using PlainTabs rather than MooTabs. This document serves as both a sample implementation and the distributed README for the PlainTabs extension to MooTools.

Installation

Just copy the plaintabs.js file into the appropriate folder of your web site. Then include the following script call in the head area of your web page:

<script src="plaintabs.js" type="text/javascript" charset="utf-8"></script>

Just make sure you correct the path for the src attribute.

That should be all that's needed to install the script, check out the implementation tab for what to do once you have installed the script.

HTML Structure

You should create your tab set using the following structure:

<div id="wrapper">
  <ul>
    <li>Tab 1</li>
    <li>Tab 2</li>
    <li>Tab 3</li>
  </ul>
  <div>Panel 1</div>
  <div>Panel 2</div>
  <div>Panel 3</div>
</div>

The id attribute on the wrapper div is required and will be used as the handle to get a hold of the whole tab set. You should notice that compared to some other tab implementations there are not superfluous CSS classes or identifiers, just the one for the tab set. You can put whatever other markup you like inside the li and div tags, just do not fiddle with the basic structure inside the wrapper div.

Once you have the basic HTML structure down you can then actually setup the tabbed behavior by adding the following code within your web page's head area:

<script type="text/javascript" charset="utf-8">
//<![CDATA[
  window.addEvent('domready', function() {
    var tabSet = new PlainTabs("wrapper");
  });
//]]>
</script>

Beyond that all you need to do is style the tab ul and div elements appropriately. Unless you tweak the defaults you can also define special styles for the active li and div areas.

To find out more about tweaking the default behavior check out the Advanced Features tab.
