Markdown Toolbar for MooTools!


markdown_toolbar.js

I am using the fantastic javascript framework MooTools for most of my projects. I need a sweet and simple way for my clients to add Markdown markup, but the awesome Control.TextArea was sadly not compatible with MooTools.

The reason for this is that Prototype.js and MooTools doesn't play well with each other (method overloading issues for example)

So after a few hours of fiddling around, here is the first version of markdown_toolbar.js!

  • lists will arrive later.. it's 5:am here in Sweden
